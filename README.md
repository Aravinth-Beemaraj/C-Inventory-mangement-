# C-Inventory-mangement-
//Made by Aravinth 

About this project
Overview
This project is a basic Inventory Management System implemented in C. It allows users to manage stock items, including adding, deleting, displaying, and calculating the total inventory amount.

Features
1.Add Stock: Add new items to the inventory with details such as item code, name, quantity, and price.

2.Delete Stock: Remove items from the inventory based on their item code.

3.Display Stock: View the current inventory stock, including item code, name, quantity, and price.

4.Calculate Total Amount: Obtain the total monetary value of the entire inventory.
